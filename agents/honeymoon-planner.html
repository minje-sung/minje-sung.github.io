新婚旅行
プランナー Honeymoon Trip
Planner 신혼여행
플래너

目的 Purpose 목표

OpenAI Agents SDKのチュートリアルをベースに構築した入門エージェント。カップルの希望・予算・日程をヒアリングし、行き先・宿・アクティビティまで含む最適な新婚旅行プランを自律的に提案します。 A tutorial-based introductory agent built with the OpenAI Agents SDK. Listens to the couple's wishes, budget, and schedule to autonomously propose a complete honeymoon plan — destination, hotels, and activities included. OpenAI Agents SDK 튜토리얼을 베이스로 구축한 입문 에이전트. 커플의 요구사항·예산·일정을 청취하여 목적지·숙소·액티비티를 포함한 최적의 신혼여행 플랜을 자율적으로 제안합니다.

協業エージェント Agent Roles 협업 에이전트
🎤 ヒアリングエージェント 🎤 Requirement Agent 🎤 요구사항 에이전트
カップルの希望・予算・旅行スタイルをインタラクティブに収集する。 Interactively collects the couple's wishes, budget, and travel style. 커플의 희망사항·예산·여행 스타일을 인터랙티브하게 수집합니다.
🗺️ プランナーエージェント 🗺️ Planner Agent 🗺️ 플래너 에이전트
収集した要件をもとに、行き先・ホテル・アクティビティを調査・選定して旅行プランを策定する。 Researches and selects destinations, hotels, and activities based on collected requirements to build a travel plan. 수집된 요구사항을 바탕으로 목적지·호텔·액티비티를 조사·선정하여 여행 플랜을 수립합니다.
📄 レポートエージェント 📄 Report Agent 📄 리포트 에이전트
プランを日程表・費用見積もり・注意事項を含む読みやすいレポートに整形して出力する。 Formats the plan into a readable report with itinerary, cost estimate, and travel tips. 플랜을 일정표·비용 견적·주의사항을 포함한 읽기 쉬운 리포트로 정형화하여 출력합니다.
