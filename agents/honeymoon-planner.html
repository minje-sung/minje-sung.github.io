目的 Purpose 목표

OpenAI Agents SDKで構築したマルチエージェント旅行プランニングアシスタント。ルーターエージェントが航空券・ホテル・スケジュールの専門エージェントへタスクを委譲し、入力ガードレールで無関係な質問をブロック。MCPサーバーと連携したリアルタイム検索でチャット形式の新婚旅行プランを提案します。 A multi-agent travel planning assistant built with the OpenAI Agents SDK. A router agent delegates tasks to specialized agents for flights, hotels, and itineraries. An input guardrail filters off-topic queries, while MCP server integration enables real-time search — all delivered through a conversational chat UI. OpenAI Agents SDK로 구축한 멀티 에이전트 여행 플래닝 어시스턴트. 라우터 에이전트가 항공·호텔·일정 전문 에이전트에게 태스크를 위임하고, 입력 가드레일로 무관한 질문을 차단합니다. MCP 서버 연계 실시간 검색으로 채팅 형식의 신혼여행 플랜을 제안합니다.

エージェント構成 Agent Roles 에이전트 구성
🧭 ルーターエージェント 🧭 Router Agent 🧭 라우터 에이전트
ユーザーの意図を解析し、適切な専門エージェントへHandoffする司令塔。入力ガードレールで旅行無関係な質問を事前ブロックし、不要なAPI消費を削減する。 Analyzes user intent and hands off to the appropriate specialist agent. An input guardrail pre-filters off-topic questions to minimize unnecessary API costs. 사용자 의도를 분석하고 적절한 전문 에이전트로 Handoff하는 사령탑. 입력 가드레일로 여행과 무관한 질문을 사전에 차단하여 불필요한 API 비용을 절감합니다.
✈️ フライトエージェント ✈️ Flight Agent ✈️ 플라이트 에이전트
flights-MCPサーバーと連携し、条件に合う航空便をリアルタイムで検索・提案する。 Connects to the flights-MCP server to search and suggest real-time flight options matching the user's conditions. flights-MCP 서버와 연계하여 조건에 맞는 항공편을 실시간으로 검색·제안합니다.
🏨 ホテルエージェント 🏨 Hotel Agent 🏨 호텔 에이전트
Playwright MCPでGoogleマップをブラウザ自動操作し、周辺情報を含むホテル候補をリアルタイムで取得する。 Uses Playwright MCP to automate Google Maps in a real browser, fetching hotel candidates with live local information. Playwright MCP로 Google Maps를 브라우저 자동 조작하여 주변 정보가 포함된 호텔 후보를 실시간으로 취득합니다.
🗓️ スケジュールエージェント 🗓️ Schedule Agent 🗓️ 스케줄 에이전트
航空・宿泊の確定情報を統合し、日程・アクティビティを組んだ旅程をPlan Boardにリアルタイム表示する。 Integrates confirmed flights and hotel data to build a full itinerary with activities, displayed live on the Plan Board sidebar. 항공·숙박 확정 정보를 통합하여 일정·액티비티가 포함된 여정을 Plan Board에 실시간으로 표시합니다.
