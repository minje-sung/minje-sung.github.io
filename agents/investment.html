에이전트

目的 Purpose 목표

市場データをリアルタイムで収集・分析し、ユーザーの投資スタイルや目標に合わせたパーソナライズされた戦略レポートを生成。教育・学習目的のマルチエージェントシステムです。 Collects and analyzes market data in real-time to generate personalized investment strategy reports tailored to the user's style and goals. A multi-agent system for educational purposes. 시장 데이터를 실시간으로 수집·분석하여 사용자의 투자 스타일과 목표에 맞는 개인화된 전략 리포트를 생성합니다. 교육·학습 목적의 멀티에이전트 시스템입니다.

協業エージェント Agent Roles 협업 에이전트
📊 マーケットデータエージェント 📊 Market Data Agent 📊 시장 데이터 에이전트
株価・為替・経済指標などのリアルタイムデータを収集・整形。 Collects and structures real-time data including stock prices, exchange rates, and economic indicators. 주가·환율·경제지표 등의 실시간 데이터를 수집·정형화합니다.
🧠 分析エージェント 🧠 Analysis Agent 🧠 분석 에이전트
セッションメモリを活用してユーザープロファイルを記憶し、リスク許容度・投資期間に基づいて市場データを分析。 Uses session memory to retain user profile and analyzes market data based on risk tolerance and investment horizon. 세션 메모리를 활용해 사용자 프로필을 기억하고, 리스크 허용도·투자 기간을 기반으로 시장 데이터를 분석합니다.
📋 戦略レポートエージェント 📋 Strategy Report Agent 📋 전략 리포트 에이전트
分析結果をもとに、ポートフォリオ提案・リスク評価・アクションプランを含むレポートを生成。 Generates a report with portfolio suggestions, risk assessment, and action plans based on analysis results. 분석 결과를 바탕으로 포트폴리오 제안·리스크 평가·실행 계획을 포함한 리포트를 생성합니다.
