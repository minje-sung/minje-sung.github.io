目的 Purpose 목표

AIに関するニューストピックを入力するだけで、収集・スコアリング・要約・編集まで自律的に行い、即発行可能な高品質レポートを生成する3エージェント協調システムです。 A 3-agent collaborative system that autonomously collects, scores, summarizes, and edits AI news — producing publication-ready reports from a single topic input. AI 관련 뉴스 토픽을 입력하면 수집·스코어링·요약·편집까지 자율적으로 수행하여 즉시 발행 가능한 고품질 리포트를 생성하는 3에이전트 협업 시스템입니다.

協業エージェント Agent Roles 협업 에이전트
🔍 ニュースハンターエージェント 🔍 News Hunter Agent 🔍 뉴스 헌터 에이전트
Tavily検索とWebスクレイピングでニュースを収集し、信頼性・関連性スコアを付与。 Collects news via Tavily search and web scraping, scoring each article for relevance and credibility. Tavily 검색과 웹 스크래핑으로 뉴스를 수집하고 신뢰도·관련성 점수를 부여합니다.
📝 要約エージェント 📝 Summary Agent 📝 요약 에이전트
選別された記事から核心テーマと統計データを抽出し、詳細要約を作成。 Extracts key themes and statistics from selected articles to produce detailed summaries. 선별된 기사에서 핵심 주제와 통계 데이터를 추출하여 상세 요약본을 작성합니다.
✨ キュレーターエージェント ✨ Curator Agent ✨ 큐레이터 에이전트
要約をもとに、主要記事・速報・編集者分析・引用情報を含む最終レポートを完成。 Compiles summaries into a final report with headlines, breaking news, editorial analysis, and citations. 요약본을 바탕으로 주요 기사·속보·편집자 분석·인용 정보를 포함한 최종 보고서를 완성합니다.
