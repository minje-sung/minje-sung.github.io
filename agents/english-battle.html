目的 Purpose 목표

ゲーム形式で英作文スキルを鍛えるシステム。AIと対戦しながら採点・フィードバック・ランキングを通じて、楽しく英語力を向上させます。チェックポイント機能で対戦履歴を保持します。 A game-based system to sharpen English writing skills. Compete against AI with real-time scoring, detailed feedback, and rankings to make language learning engaging. Checkpointing preserves battle history. 게임 형식으로 영작문 실력을 키우는 시스템. AI와 대결하면서 채점·피드백·랭킹을 통해 즐겁게 영어 실력을 향상시킵니다. 체크포인팅으로 대전 기록을 유지합니다.

協業エージェント Agent Roles 협업 에이전트
✍️ AIライターエージェント ✍️ AI Writer Agent ✍️ AI 라이터 에이전트
与えられたテーマに対して模範英作文を生成。ユーザーの回答と対比される対戦相手として機能。 Generates a model English composition on the given topic, serving as the opponent compared against the user's response. 주어진 주제에 대한 모범 영작문을 생성하며, 사용자 답안과 대비되는 상대방 역할을 합니다.
⚖️ ジャッジエージェント ⚖️ Judge Agent ⚖️ 심판 에이전트
文法・語彙・表現力・論理構成の4軸でユーザーとAIの英作文を採点し、詳細フィードバックを提供。 Scores both user and AI compositions on 4 axes — grammar, vocabulary, expression, and logic — with detailed feedback. 문법·어휘·표현력·논리 구성 4개 축으로 사용자와 AI의 영작문을 채점하고 상세 피드백을 제공합니다.
🏆 ランキングエージェント 🏆 Ranking Agent 🏆 랭킹 에이전트
チェックポイントに保存された履歴からスコア推移・弱点分析・改善提案をまとめたレポートを生成。 Generates a progress report with score trends, weakness analysis, and improvement suggestions from checkpointed history. 체크포인트에 저장된 기록으로 점수 추이·약점 분석·개선 제안을 정리한 리포트를 생성합니다.
